# TaskPro Landing Page

## 📝 Descripción

Esta es una landing page ficticia para TaskPro, una aplicación de productividad. Landing Page desarrollada utilizando tecnologías modernas y un diseño atractivo.

## ✨ Características

- **Diseño moderno:** Una interfaz limpia y profesional que presenta las funcionalidades de la aplicación.
- **Responsive:** Adaptable a diferentes tamaños de pantalla, desde computadoras de escritorio hasta dispositivos móviles.
- **Secciones informativas:** Incluye secciones sobre características, testimonios, etc.

## 🛠️ Tecnologías utilizadas

- HTML
- TypeScript
- Vue.js
- SCSS
- Vercel
